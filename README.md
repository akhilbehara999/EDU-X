By leveraging cutting-edge AI and machine learning technologies, we're creating an immersive learning environment where students can master not just languages, but complex subjects across disciplines, all in their most comfortable linguistic medium.

## Problem Statement

**Multilingual Smart Education Web/Mobile Platform**

A web/mobile application designed to help students understand and learn content in non-native languages. The platform supports multilingual content and provides real-time translation of text, audio, and video materials. It features adaptive language learning tools that adjust difficulty based on the student's proficiency, interactive vocabulary exercises, and comprehension quizzes. Additional features include contextual word explanations, pronunciation guides, progress tracking, personalized learning paths, and collaborative discussion forums where students can ask questions or clarify doubts in their preferred language. By combining translation, adaptive learning, and interactive tools, the platform enhances comprehension and makes learning accessible and engaging for students across different language backgrounds.

## Local Setup Requirements

### Prerequisites
- Node.js (version 16 or higher)
- npm (comes with Node.js)
- A modern web browser (Chrome, Firefox, Edge, Safari)

### Installation Steps

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd hiii
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_actual_api_key_here
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Access the application**:
   Open your browser and navigate to `http://localhost:3000` (or the port shown in your terminal)

### Build for Production
To create a production build:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```
